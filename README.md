# Backend VoiceChat

- Se esta usando MongoDB Atlas para base de datos.

Para correr el proyecto se debe seguir los siguientes pasos:
- Instalar las librerias necesarias ``npm i``
- Correr el proyecto en local usando nodemon ``npm run dev``
